# Тестовое задание ПЭСК

### Демо проекта
Ознакомиться с демо можно по [этой ссылке](https://olegus199.github.io/test-toast/)

### Запуск проекта
1. Клонируйте проект
```bash
git clone https://github.com/olegus199/test-toast.git
```
2. Перейдите в директорию проекта
```bash
cd test-toast
```
3. Установите зависимости
```bash
pnpm i
```
или
```bash
npm i
```
4. Запустите проект
```bash
pnpm run dev
```
или
```bash
npm run dev
```
5. Перейдите по адресу
```
http://localhost:5173
```
5. Запуск тестов
```bash
pnpm run test
```
